Understanding the CSS Box Model

The CSS Box Model describes how elements are structured and sized on a web page.

  • Content: The actual text or images inside the box.
  • Padding: The space around the content, inside the border.
  • Border: The line surrounding the padding and content.
  • Margin: The space outside the border, separating this box from others.
Margin (30px) — space outside the border
Border (5px solid teal) — surrounds padding + content
Padding (20px) — space between border and content
Content area (300px wide) — This is where your text or images go.
